# 📝 Notic

*Notic* is a simple and responsive *Notes Web App* built with *Next.js*.  
It allows users to *add, edit, delete, and search notes* easily, with a smooth and modern user experience powered by *Framer Motion* animations.

---

## 🚀 Features

- ⚡ **Built with [Next.js](https://nextjs.org/)** for fast performance and scalability  
- 🎞 *Smooth animations* using [Framer Motion](https://www.framer.com/motion/)  
- 📱 *Fully responsive* design for all devices  
- 🗒 *Add, edit, delete, and search* your notes in real time  
- 💾 *Server actions* for seamless CRUD operations  
- 🔍 *Search functionality* to quickly find your notes  

---

## 🛠 Tech Stack

- *Framework:* Next.js  
- *Animation:* Framer Motion  
- *Language:* TypeScript / JavaScript  
- *Styling:* Tailwind CSS  

---

## 💡 How It Works

1. Write and save a new note.  
2. Edit or delete existing notes with instant feedback.  
3. Use the *search bar* to find notes by keywords.  
4. All operations update instantly without a full page reload.

## 🏃‍♂ Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/notic.git

2. Install dependencies:

3. Run the development server:

npm run dev


4. Open http://localhost:3000 in your browser.
